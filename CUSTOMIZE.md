# Customize before publishing

This repository is designed to work as both a GitHub profile README and a GitHub Pages portfolio.

1. Name the repository exactly the same as your GitHub username to activate the profile README.
2. The repository is configured for the GitHub account `kbsybf`; replace the display name in `README.md` and `docs/index.html` if desired.
3. Replace the research interests, toolbox, and featured work with your real experience.
4. Add links to your strongest repositories, papers, videos, datasets, and contact channel.
5. GitHub Pages is configured to publish directly from the `docs` folder on the `main` branch.

The site has no build step or third-party JavaScript dependency. It can also be previewed by opening `docs/index.html`.
